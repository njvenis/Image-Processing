# Image-Processing
Using MATLAB to process, find and categorise various images.

# Transformations and processes included:
- Bilinear Interpolation
- NN Interpolation
- Grey level slicing
- Guassian Filtering
- Bilinear Filtering
- Manual Thresholding
- Otsu's Thresholding
- Opening and Closing (Morphology)
- Canny and Sobel Edge Detection
- Object recognition
